# Terminal Tetris

A simple implementation of **Tetris in the terminal using Python**.

The game runs entirely in a text-based terminal and renders graphics using ASCII characters.

## Requirements

* Python 3

## Running the Game

To start the game, run:

```bash
python3 main.py tetris
```

## Controls

* **← / →** : Move piece left or right
* **↓** : Move piece down
* **A** : Rotate left
* **Z** : Rotate right

After a game over, press **A** or **Z** to start a new game.
